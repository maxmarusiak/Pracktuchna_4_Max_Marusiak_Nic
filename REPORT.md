## 1. Незмінність рядків (immutability) у C# та роль StringBuilder
string у C# є незмінним типом.  
Це означає, що будь-яка операція над рядком (конкатенація, Replace, Insert, Remove) створює новий об’єкт у пам’яті, а старий залишається непотрібним.

## 4. Які методи класу String використовувалися найбільше
Основні методи:
Substring — виділення частини рядка

IndexOf / LastIndexOf — пошук фрагментів

Replace — заміна підрядків

Split — розбиття тексту на слова/речення

Join — формування CSV

Trim — очищення пробілів

ToLower / ToUpper — нормалізація регістру
